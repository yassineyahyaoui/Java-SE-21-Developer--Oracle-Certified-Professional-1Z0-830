# Module 01 -- Introduction to Java SE 21 Developer Certification Course

### Q: What is the structure of the Java SE 21 Developer Professional exam (1Z0-830)?
The exam has 50 multiple-choice questions, a duration of 120 minutes, and requires 68% to pass—so you must answer at least 34 questions correctly.

### Q: What are the key Java 21 topics covered on the 1Z0-830 exam?
Pattern matching (instanceof and switch), record patterns, sealed types, records, and virtual threads are among the Java 21 additions emphasized on the exam. The course aligns with the official exam objectives for these and core Java SE topics.

### Q: What was removed from the Java 17 exam that is not on the 1Z0-830?
JDBC was removed; the 1Z0-830 focuses on core Java SE 21 without JDBC.
