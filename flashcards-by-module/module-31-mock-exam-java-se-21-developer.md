# Module 31 -- Mock Exam - Java SE 21 Developer Certification

### Q: What is the purpose of the mock exam in this course?
It simulates the real 1Z0-830 exam: 50 questions, 120 minutes, multiple-choice. Use it to practice under timed conditions and identify weak areas before the actual exam.

### Q: What Java 21 features are emphasized on the 1Z0-830?
Pattern matching (instanceof and switch), record patterns, sealed types, records, and virtual threads. The exam aligns with Oracle's Java SE 21 Developer exam objectives.

### Q: How should you use the mock exam result?
Review every incorrect answer; understand why the correct option is right. Re-read the relevant objective and practice with code. Target 68% or higher (34+ correct) to be ready for the real exam.
